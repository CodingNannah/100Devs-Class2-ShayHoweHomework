Dedicated to the Craft of Building Functional and Beautiful Websites

Every year the brightest web designers and front-end developers unite to discuss the latest technologies.

In 2023, Diana Barber joined them. She embarked on a Fullstack JavaScript software engineering journey with Leon Noel's #100Devs. Day Two homework included Shaye Howe's 12-lesson Learn to Code HTML & CSS.

Shaye hosts a "Styles Conference" in which newbies learn the fundamentals of HTML & CSS by building a website, as if they were coding along at a web development conference.
